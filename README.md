# EcoCast Renewable Energy Analytics Dashboard

## Overview

EcoCast is an end-to-end weather analytics and renewable energy intelligence platform that integrates API-driven data collection, ETL processing, PostgreSQL data storage, and interactive business intelligence visualization.

The project extracts weather forecast data from the Open-Meteo API, performs transformation and validation through a production-style ETL pipeline, stores analytics-ready datasets in PostgreSQL, and delivers actionable renewable energy insights through an interactive Dash dashboard.

This project demonstrates modern data engineering and analytics workflows including API integration, ETL development, database management, and dashboard deployment.

## Business Problem

Renewable energy planning depends heavily on weather conditions.

Organizations and decision-makers need a simple way to evaluate:

* Temperature trends
* Wind speed forecasts
* Cloud cover conditions
* Solar energy potential
* Wind energy potential

EcoCast transforms raw weather forecasts into actionable renewable energy recommendations.

## Installation

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/ecocast-renewable-energy-dashboard.git
cd ecocast-renewable-energy-dashboard
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Configure PostgreSQL

Update the database connection string in:

```python
DATABASE_URL
```

Example:

```python
postgresql://postgres:password@localhost:5433/ecocast_db
```

---

## Run the ETL Pipeline

```bash
python ecocast_etl.py
```

This process:

* Extracts weather data
* Cleans and validates records
* Calculates renewable energy metrics
* Loads data into PostgreSQL

---

## Launch the Dashboard

```bash
python app.py
```

Open:

